# Scoring

## Pondération globale

- Fondamental : 30 %
- Dividende : 25 %
- Valorisation : 15 %
- Technique : 15 %
- Actualites : 10 %
- Risque : 5 %

## Interpretation

- 0-39 : eviter
- 40-54 : faible interet
- 55-69 : surveiller
- 70-79 : opportunite correcte
- 80-89 : opportunite forte
- 90-100 : exceptionnel, validation manuelle obligatoire

## Probabilites

Le MVP utilise une approche heuristique prudente, bornee entre 5 % et 95 %, basee sur les scores technique, fondamental, valorisation, actualites et risque de dividende.
